## 📅 Day 02: Logic, String Slicing & Control Flow
Date: February 27, 2026

Track: Data Analyst Journey

### 📝 Concepts covered
Today’s focus was on Programmatic Decision Making and Data Extraction. I moved beyond basic variables to understand how Python evaluates conditions and manipulates string data.

### 1. String Slicing & Metadata
Slicing: Learned to extract specific segments using variable[start:end] syntax.

Data Inspection: Using type() to verify data categories (int, float, str) before processing.

### 2. Type Conversion (Type Casting)
Mastered explicit conversion using int(), float(), str(), and bool().

Identified ValueError risks when attempting to convert non-numeric strings to integers.

### 3. Operators (Relational & Logical)
Relational: Used >, <, ==, !=, >=, <= to compare values.

Logical: Combined multiple conditions using and, or, and not.

### 4. Control Flow (Conditional Statements)
Implemented if-else blocks to create branching logic.

Key Learning: Mastered Python's Indentation rules (4-space standard) to define code blocks.
