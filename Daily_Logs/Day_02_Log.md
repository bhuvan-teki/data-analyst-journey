### 📝 Concepts Covered
Today’s focus was on Programmatic Decision Making and Data Extraction. I moved beyond basic variables to understand how Python evaluates conditions and manipulates string data.

### 1. String Slicing & Metadata
Slicing: Learned to extract specific segments using variable[start:end] syntax.

Data Inspection: Using type() to verify data categories (int, float, str) before processing.

### 2. Type Conversion (Type Casting)
Explicit Conversion: Mastered the use of int(), float(), str(), and bool() to ensure data integrity.

Error Handling: Identified ValueError risks when attempting to convert non-numeric strings to integers.

Relational: Used >, <, ==, !=, >=, <= to perform value comparisons.

Logical: Combined multiple conditions using and, or, and not to build complex filters.

### 4. Control Flow (Conditional Statements)
Branching Logic: Implemented if-else blocks to create dynamic program paths.

Indentation: Mastered Python's 4-space standard to define code blocks—a critical skill for clean, readable code.
